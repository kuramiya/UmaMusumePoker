# ルール
- CPUと一対一
- 全カード、キャラごとに2枚ずつ
- 手持ちは5枚、1ターンで入れ替えていく
- 同じ山札から取る
- 捨てられたカードは確認可能
- ジュエル10からスタート
- 掛け数は基本1、レイズ可能
- 勝負は5ターン
