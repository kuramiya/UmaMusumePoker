# クローズド・ポーカーのルール

1. クローズド・ポーカー
1. ゲームに参加するプレイヤーはアンティを払う。
1. ディーラーはアンティを払ったプレイヤーに5枚ずつ裏向きにカードを配り、残りの札は中央に裏向きに置く。
1. 配り終わったらプレイヤー達はカードを手に持つ。
1. 1度目のベッディング・インターバル。
1. 各アクティブ・プレイヤーは一度だけカード交換を行う。ディーラーから順に任意の枚数（0枚も可）のカードを場に裏向き捨て、捨てた枚数と同じ枚数のカードを中央のパイルから取る。
1. 2度目のベッディング・インターバル。
1. アクティブ・プレイヤーは自分の手札を公開する。
1. 勝者はポットのチップを全て取る。

# ルール

- ビデオポーカーとする
- プレイヤーのみ
- 全カード、キャラごとに2枚ずつ
- 手札は5枚
- ジュエル10からスタート
- 必ず1ジュエルをかける必要あり
- かけられるジュエルは制限なし
- ジュエルがゼロになったら終了

# 仕様・設計

## カード枚数

- 83枚 x 2 = 166枚
- キャラが増えているので、もうちょっと増えるはず

## ゲーム進行（ビデオポーカーの場合）

- かけるジュエルを設定する
- 最初に全カードのランダム配列を作成する（山札）
- ユーザーの入力待ち（１回目）
  - ホールドするカードを選ぶ
- カードの入れ替え
- ユーザーの入力待ち（２回目）
  - ホールドするカードを選ぶ
- カードの入れ替え
- 結果の決定、精算

- ゲーム性に乏しい

## ゲーム進行（クローズド・ポーカーの場合）

- 最初に全カードのランダム配列を作成する（山札）
- 交互or連続で5枚ずつ配布する
- ユーザーの入力待ち（１回目）
  - ベット（ジュエルをかける　１個～）
  - ホールドするカードを選ぶ
  - 降りるor進む
- CPUが同様の処理を行う
- ユーザー、CPUのカードの入れ替えを行う
- 入れ替えされたカードを場に表示する
- ユーザーの入力待ち（２回目）
- CPUが同様の処理を行う
- 勝ち負け確認
  - ドローの場合、お互いベット没収
- 次のゲームに進む
- 5ゲーム終わった後の成績で勝ち負けが決まる

## CPU設計（クローズド・ポーカーの場合）

- 戦略をいくつか用意する
  - 手堅い
    - 揃った役は手放さない
    - ベットを増やさない
    - 相手がベットを増やすと降りる
  - 普通
    - 揃った役は手放さない
    - 役に応じてベットを増やす（一定ではなく確率に応じて）
    - 相手がベットを大きく増やすと、降りる
  - 暴挙
    - 役を無視する
    - ベットは確率で増やす
    - 降りない
- ゲーム毎に戦略を変える
