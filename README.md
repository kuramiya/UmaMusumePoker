# UmaMusumePoker
うまぽーかー

# ルール

- CPUと一対一
- 全カード、キャラごとに2枚ずつ
- 手持ちは5枚、1ターンで入れ替えていく
- 同じ山札から取る
- 捨てられたカードは確認可能
- ジュエル10からスタート
- 1ゲーム5回

# 仕様・設計

## カード枚数

83枚 x 2 = 166枚

## ゲーム進行

- 最初に全カードのランダム配列を作成する（山札）
- 交互or連続で5枚ずつ配布する
- ユーザーの入力待ち（１回目）
  - ベット（ジュエルをかける　１個～）
  - ホールドするカードを選ぶ
  - 進む
- CPUが同様の処理を行う
- ユーザー、CPUのカードの入れ替えを行う
- 入れ替えされたカードを場に表示する
- ユーザーの入力待ち（２回目）
- CPUが同様の処理を行う
- 勝ち負け確認
  - ドローの場合、お互いベット没収
- 次のゲームに進む
- 5ゲーム終わった後の成績で勝ち負けが決まる

## CPU設計

- 戦略をいくつか用意する
  - 手堅い
    - 揃った役は手放さない
    - ベットを増やさない
  - 普通
    - 揃った役は手放さない
    - 役に応じてベットを増やす（一定ではなく確率に応じて）
  - 暴挙
    - 役を無視する
    - ベットは確率で増やす
- ゲーム毎に戦略を変える
